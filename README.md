# Glif

This was initially intended to be a AI project but has become a mish-mash of graphical ideas using CSS and JavaScript and a collection of algorithms written in C and JavaScript.

The main compartments of this repo are,

- [Algorithms](./algos/)
- [Autotools](./autotools/)
- [Canvas API](./canvas/)
- [CSS](./css3/)
